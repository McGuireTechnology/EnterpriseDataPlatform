# Policy Templates

This folder is reserved for Markdown policy templates converted from contrib source documents.

Keep converted templates text-first and fragment-friendly so EDP can map specific sections to controls, safeguards, objectives, evidence requirements, findings, and posture changes.

## Priority CIS Templates

Files prefixed with `cis-` are converted from CIS Controls v8.1 policy templates in `contrib/sources/cis/Policy Templates`. These should be treated as the priority policy-template set because their front matter includes official CIS Control mappings when the source template identifies a mapped control.
